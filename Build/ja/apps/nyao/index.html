Nyao Sketch

スケッチ、キャンバス、プレゼンテーション

簡単なスケッチから会議・プレゼン中のリアルタイム注釈まで、Nyao Sketchで画面上にそのまま描き込めます。

Key features

    macOS専用アプリ

Tech stack

AppKit SwiftUI Metal CoreGraphics

画面上に直接描ける、リアルタイムスケッチツール「Nyao Sketch」

シンプルなスケッチから会議・プレゼン中のライブ注釈まで、Nyao Sketchなら画面上にそのまま描き込めます。
Nyao Sketchは透明キャンバスを使って、ほかのアプリの上から自然に描画できるスケッチ／注釈アプリです。ドキュメント、Web、デザインカンプ、スライドを見ながら要点をすぐに強調・説明でき、Metalベースのレンダリングにより、軽快なレスポンスと滑らかなストロークを実現します。

主な機能

  • 透明キャンバスオーバーレイ：ほかのアプリに重ねて描く感覚で作業でき、キャンバスの透明度も自由に調整

  • ウィンドウの素早い表示／非表示：必要なときにすぐ表示し、不要なときはすっきり隠せる

  • ドローモード／クリックスルーモード：シーンに合わせてキャンバス入力方式を切り替え

  • カスタムブラシ登録：よく使うブラシを自分の作業スタイルに合わせて構成

  • ファイル保存・PNG書き出し：作業結果を手軽に保存・共有

  • ショートカットのカスタマイズ：一部ショートカットを好みのキーに変更可能

こんなシーンで活躍します

  • 会議・プレゼン中、画面共有しながら要点をその場で示して説明したいとき

  • ブラウザ・ドキュメント・デザインツール上にチェックやマーキング、ラフな書き込みで素早くフィードバックしたいとき

  • リファレンスを見ながら、透明キャンバスでトレースしたり構図を素早く検討したいとき
